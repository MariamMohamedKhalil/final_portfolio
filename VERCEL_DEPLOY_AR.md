# دليل النشر على Vercel

## الطريقة الأولى: النشر عبر موقع Vercel (الأسهل)

1. اذهب إلى [vercel.com](https://vercel.com)
2. سجل دخول باستخدام حساب GitHub
3. اضغط على "Add New Project"
4. اختر المستودع (Repository) الخاص بك
5. اضغط "Deploy"

## الطريقة الثانية: النشر عبر سطر الأوامر

### التثبيت
```bash
npm i -g vercel
```

### تسجيل الدخول
```bash
vercel login
```

### النشر
```bash
vercel
```

### النشر للإنتاج
```bash
vercel --prod
```

## الملفات المهمة

- ✅ `vercel.json` - إعدادات Vercel
- ✅ `.vercelignore` - الملفات المستبعدة من النشر
- ✅ `package.json` - معلومات المشروع

## ملاحظات مهمة

1. جميع الملفات جاهزة للنشر
2. لا يوجد خطوة بناء (build) مطلوبة
3. الموقع ثابت (static site)
4. جميع المسارات نسبية وصحيحة

## بعد النشر

سيكون موقعك متاح على:
- `https://your-project.vercel.app`

## إضافة دومين خاص

1. اذهب إلى إعدادات المشروع في Vercel
2. اختر "Domains"
3. أضف الدومين الخاص بك
4. اتبع التعليمات لتحديث DNS

## استكشاف الأخطاء

إذا واجهت مشاكل:
1. تحقق من console المتصفح
2. تأكد من رفع جميع الصور والفيديوهات
3. راجع سجلات النشر في Vercel
